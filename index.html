Lesson 01 — Tags

What is an HTML tag?

An HTML tag is an instruction wrapped in angle brackets, like <p>. It tells the browser what a piece of content is — a paragraph, a heading, a button — so the browser knows how to show it.

Most tags come in pairs: an opening tag <p> and a closing tag </p>, with your content sandwiched in between.
